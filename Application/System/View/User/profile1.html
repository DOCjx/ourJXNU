广州名航技术有限公司

中国民航信息集团公司正式组建于2002年10月，属国资委管理的中央企业。中国民航信息网络股份有限公司是在2000年10月，由中国民航计算机信息中心联合当时所有国内航空公司发起成立，2001年2月在香港联交所主板挂牌上市交易，股票代码为0696.HK。2008年7月，中国民航信息集团公司以中国民航信息网络股份有限公司为主体，完成主营业务和资产重组并在香港成功整体上市。公司实收资本19.5亿元人民币，资产总额105亿元人民币。现有员工4300多人，总部设在北京。（公司名称以下简称中国航信）
